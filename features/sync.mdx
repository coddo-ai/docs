Coddo offers **bidirectional sync** between the desktop app and the web dashboard. Your projects, tasks, and chats stay in sync across all your devices.

## How it works

The sync system keeps your data consistent between:

- **Desktop app** — Your local machine (filesystem is the source of truth)
- **Web dashboard** — Cloud-based at [coddo.ai](https://coddo.ai)

When you make changes on either side, they're synchronized automatically.

## What syncs

| Data | Description |
|------|-------------|
| **Projects** | Project metadata, settings, and tags |
| **Tasks** | Kanban tasks with all their metadata |
| **Chats** | Chat threads and message history |

## Sync status

The sync status is visible in the app, showing you:

- **Synced** — All data is up to date
- **Syncing** — Changes are being transferred
- **Conflict** — Changes were made on both sides and need resolution

## Requirements

To use sync, you need:

1. A **Coddo account** (free)
2. To be **logged in** on both the desktop app and the web dashboard
3. An **internet connection** for sync operations

<Info>Your local data is always the source of truth. If you lose internet, you can keep working — changes will sync when you reconnect.</Info>

## Web dashboard

Access your projects from any browser at [coddo.ai](https://coddo.ai). The web dashboard lets you:

- View your projects and their tasks
- Browse chat history
- Monitor project status
- Manage your account and subscription
