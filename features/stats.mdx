The Stats view gives you a **dashboard overview** of your coding activity across your project. It helps you track usage, costs, and productivity over time.

## Activity heatmap

A GitHub-style **activity heatmap** shows your daily coding activity over the last 18 weeks. Each cell represents a day, and the intensity shows how active you were.

## Daily metrics

Track key metrics for each day:

| Metric | Description |
|--------|-------------|
| **Chats** | Number of chat sessions started |
| **Messages** | Total messages exchanged with Claude Code |
| **Duration** | Total time spent in chat sessions |
| **Cost** | API usage cost in USD |
| **Tasks** | Number of tasks completed |

## Charts

Interactive charts let you visualize trends over time. Switch between different metrics:

- **Cost** — Track your spending over time
- **Messages** — See message volume trends
- **Chats** — Monitor chat session frequency
- **Tasks** — Track task completion rate

## Time granularity

View your data at two levels of detail:

- **Day** — Daily totals for trend analysis
- **Hour** — Hourly breakdown for detailed patterns

<Info>All statistics are computed from your local chat and task data. Nothing is sent to external servers.</Info>
