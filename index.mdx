## What is Coddo?

Coddo is a **free desktop application** that gives you a visual interface for [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Instead of working in the terminal, you get a beautiful chat UI and an integrated Kanban board to track task execution in real time.

<CardGroup cols={3}>
  <Card title="Chat Interface" icon="message-bot">
    A native chat UI to interact with Claude Code — with streaming, tool use visibility, and mode selection.
  </Card>
  <Card title="Kanban Board" icon="table-columns">
    Track your tasks across columns — from backlog to done — with drag-and-drop and real-time updates.
  </Card>
  <Card title="Task Automation" icon="wand-magic-sparkles">
    Create a task, pick a branch, and let Claude Code handle the rest automatically.
  </Card>
  <Card title="Git Integration" icon="code-branch">
    Branches, commits, and merges on autopilot — all without leaving the app.
  </Card>
  <Card title="Multi-project" icon="windows">
    Open multiple projects simultaneously, each in its own window with its own chat and Kanban board.
  </Card>
  <Card title="Local & Private" icon="lock">
    Everything runs locally. Your conversations, code, and project data never leave your machine.
  </Card>
</CardGroup>

## How it works

<Steps>
  <Step title="Download & Install">
    Download Coddo for Mac or Windows from [coddo.ai](https://coddo.ai). Linux support is coming soon.
  </Step>
  <Step title="Open a project">
    Create a new project or open an existing folder. Coddo initializes a `.coddo/` directory to store your chats and tasks locally.
  </Step>
  <Step title="Chat & build">
    Create tasks on the Kanban board, chat with Claude Code through the visual interface, and watch your tasks progress in real time.
  </Step>
</Steps>
