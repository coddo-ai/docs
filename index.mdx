## What is Coddo?

Coddo is a **free desktop application** that gives you a visual interface for [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Instead of working in the terminal, you get a beautiful chat UI and an integrated Kanban board to track task execution in real time.

<CardGroup cols={2}>
  <Card title="Chat with Claude Code" icon="message-bot">
    A rich chat interface with markdown rendering, diff viewer, plan approval, and image support.
  </Card>
  <Card title="Kanban Board" icon="table-columns">
    Track your tasks across columns — from backlog to done — with drag-and-drop and real-time updates.
  </Card>
  <Card title="Git Integration" icon="code-branch">
    Manage branches, stage changes, commit, push, and handle merge conflicts — all from the UI.
  </Card>
  <Card title="100% Free" icon="gift">
    All features are included for free. No credit card required, no time limits.
  </Card>
</CardGroup>

## How it works

<Steps>
  <Step title="Download & Install">
    Download Coddo for Mac or Windows from [coddo.ai](https://coddo.ai).
  </Step>
  <Step title="Open a project">
    Create a new project or open an existing folder. Coddo initializes a `.openspace/` directory to store your chats and tasks locally.
  </Step>
  <Step title="Chat & build">
    Create tasks on the Kanban board, chat with Claude Code through the visual interface, and watch your tasks progress in real time.
  </Step>
</Steps>
