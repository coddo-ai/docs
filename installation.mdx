## Prerequisites

Before installing Coddo, make sure you have **Claude Code** installed on your machine. Claude Code is a CLI tool from Anthropic that Coddo uses under the hood.

Install Claude Code using one of the following methods:

<Tabs>
  <Tab title="npm">
    ```bash
    npm install -g @anthropic-ai/claude-code
    ```
  </Tab>
  <Tab title="Homebrew">
    ```bash
    brew install claude-code
    ```
  </Tab>
  <Tab title="Native install">
    Download the installer from the [official Claude Code page](https://docs.anthropic.com/en/docs/claude-code). This method does not require Node.js.
  </Tab>
</Tabs>

Once installed, verify it works:

```bash
claude --version
```

Then log in to your Anthropic account:

```bash
claude
```

Follow the prompts to authenticate. You need an active Claude subscription (Claude Pro, Team, or Enterprise) to use Claude Code.

## Download Coddo

<Card title="macOS" icon="apple">
  Download the `.dmg` file from [coddo.ai](https://coddo.ai).

  1. Open the `.dmg` file
  2. Drag **Coddo** into your **Applications** folder
  3. Open Coddo from Applications

  <Tip>If macOS blocks the app, go to **System Settings → Privacy & Security** and click **Open Anyway**.</Tip>
</Card>

<Info>Windows and Linux versions are coming soon.</Info>

## First launch

When you open Coddo for the first time, you'll go through a quick onboarding:

1. **Welcome screen** - Overview of Coddo
2. **Claude Code check** - Coddo verifies that Claude Code is installed and accessible
3. **Auth check** - Coddo verifies that Claude Code is authenticated
4. **Login** - Sign in with your Coddo account (Google or email/password)
5. **Ready** - You're all set to start using Coddo

<Info>Creating a Coddo account is free. All your project data stays local on your machine.</Info>

## Updates

Coddo checks for updates automatically. When a new version is available, you'll see a banner in the app. You can also check manually from **Settings → Check for updates**.
